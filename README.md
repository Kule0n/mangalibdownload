---
♦ Инструкция скрипта:
1. Распаковать архив в папку
2. Перейти в папку с помощью консоли и запустить скрипт `php index.php URL`

```
php index.php https://mangalib.me/higurashi-no-naku-koro-ni-hirukowashi-hen
```
